# How to contribute to a project in GitHub?
## 0. Download Git.exe
##### 1. Go to this website (https://git-scm.com/downloads) and click download.

![image](https://user-images.githubusercontent.com/23183656/31303081-28a2ee86-ab3a-11e7-8040-fffd13e69d2a.png)
##### 2. Open the setup file and install Git.exe on your computer.
##### 3. Remember to include Git Bash.

## 1. Fork the repo that you want to contribute to  
##### 1. Go to the website of the repository you want to contribute. (For this tutorial I will use Apple's Swift repo)
##### 2. Click on the `Fork` button.
![opera snapshot_2017-10-07_101717_github com](https://user-images.githubusercontent.com/23183656/31303921-07852854-ab49-11e7-83b0-b0d34809b4e4.png)
##### 3. After you click the `Fork` button, you shall see this :   


![image](https://user-images.githubusercontent.com/23183656/31303944-655d5370-ab49-11e7-807d-e1bf91b49628.png)



## 2. Clone the forked repo
##### 1. Create a new folder at Desktop and named it 'Repos'.
##### 2. Click into that folder, then right click any where, then click on 'Git Bash Here'.  
   
![image](https://user-images.githubusercontent.com/23183656/31303139-fdf33c6c-ab3a-11e7-82ea-4d42c0296a21.png)

##### 3. After that, you shall see this black console : 

![image](https://user-images.githubusercontent.com/23183656/31303154-4555cade-ab3b-11e7-9f34-ec0d924e2001.png)  

##### 4. Now, go to the link of the **FORKED** repo (not the original repo!), then click on the 'Clone or download' button : 

![image](https://user-images.githubusercontent.com/23183656/31303967-ca078188-ab49-11e7-956e-73c214ba0294.png)
 
  
##### 5. Then click on the 'Copy to clipboard' button. *(Don't click on 'Download' or 'Open in Desktop'!)* Â 
NOTE : Make sure your username is in the link, or else you might be cloning the wrong repo.

![image](https://user-images.githubusercontent.com/23183656/31303982-057cb26a-ab4a-11e7-84aa-1bbe951eb385.png)


##### 6. Now, go back to the black console and type in `git clone`, then paste the copied link.


![image](https://user-images.githubusercontent.com/23183656/31303987-3074f11c-ab4a-11e7-8494-16d4bf7b333c.png)

##### 7. Press [ENTER], then you had already cloned the project successfully.
